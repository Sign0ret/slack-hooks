# SLACK APP TEAM 46

### Clone Repo.
```
git clone https://github.com/Sign0ret/slack-hooks.git
cd slack-hooks
```
### Dev Setup.
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
### Development execution command.
```
python run.py --dev
```

### Production execution command.
```
python run.py
```
